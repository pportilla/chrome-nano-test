Setup

  • Use Chrome desktop on a supported machine.
  • Open chrome://flags/#optimization-guide-on-device-model and enable it.
  • Open chrome://flags/#prompt-api-for-gemini-nano-multimodal-input and enable it.
  • If that flag is missing, try chrome://flags/#prompt-api-for-gemini-nano.
  • Restart Chrome, then return to this page.
  • On first use, click Prepare to create the session and download Gemini Nano.
